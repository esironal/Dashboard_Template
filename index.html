Dashboard Template

Deze template is gebaseerd op het Twitter Bootstrap framework, met toevoeging van navigatie mogelijkheden voor touch devices

Navigatie

Navigatie is mogelijk door gebruik te maken van het menu links, de knoppen onder en boven het scherm, door het scrollwheel op de muis te gebruiken, of op touch devices door te swipen.

Lay-out

De lay-out is gebaseerd op de twitter bootstrap grid (12-koloms) en een custom html opmaak voor de navigatie. Door middel van copy/paste zijn er makkelijk nieuwe pagina's toe te voegen en de lokatie voor het maken van rijen staat met comments aangegeven.

Styling

Er is reeds een uitgebreid css bestand aanwezig waar de meeste standaard html elementen in staan, waardoor het uiterlijk door kleine aanpassingen snel kan worden veranderd.
